# SECIRA_SD_Classifer_Replication-NRE-Meteo-Analysis

# Исследование 21 класса ансамблевого классификатора гелиогеофизических данных SuperDARN

Репозиторий содержит материалы технологической практики по большим данным и интеллектуальным системам.

В основе проекта лежит аудит, компиляция и экспериментальный анализ существующего ансамблевого классификатора на С, разработанного ИСЗФ СО РАН (модуль `SECIRA_SD_CLASSIFIER v3.2` О. И. Бернгардта), и создание вторичного исследовательского каскада для анализа классов неопределенности.

## Воспроизводимость и Окружение
Эксперименты реализованы в среде Google Collab с использованием Intel Xeon CPU with 2 vCPUs 
* **Стек анализа данных:** Python 3.12.13, Scikit-learn 1.6.1, Pandas 2.2.2, Matplotlib 3.10.0, NumPy 2.0.2.
* **Детерминированность:** Для строгого воспроизведения многомерной кластеризации алгоритма K-Means зафиксирован `random_state = 42`.

## Результаты и Ключевые Выводы
В ходе сквозного тестирования на массиве данных ($N = 33\,870$) был изолирован Класс 21 (смесь метеорных эхо-сигналов и ближнего диффузного рассеяния NRE). Проведенный многомерный статистический анализ показал:
1. **Спектральная однородность:** Параметры скорости ($V$) и спектральной ширины ($Wl$) имеют сплошное перекрытие («кашу»), относительная разница между средними кластеров — менее 20%.
2. **Траекторная сепарация:** Алгоритм K-Means разделил выборку на две пространственные моды исключительно по геометрии волнового вектора. Наибольший вклад внесло трио признаков: `sin_k_xy_R` (40%), `cos_k_B_R` (40%) и `sin_k_xy_3R4` (34%).

**Итог исследования:** Экспериментально доказана невозможность сепарации метеоров и NRE на основе только мгновенных локальных параметров ввиду ракурсной слепоты метода на наклонных дальностях 250–450 км. Обоснована необходимость внедрения временного контекста (суточного хода) в будущей магистерской диссертации.

## Структура проекта
* `/SECIRA_SD_Classifier-main/` — компоненты базового SECIRA-классификатора и веса подсетей ансамбля.
* `SECIRA_Meteor_NRE_Classifier.ipynb` — основной рабочий ноутбук в Google Colab (сопряжение с Си-ядром, визуализация распределений, Z-масштабирование и кластеризация K-Means).
* `requirements.txt` — список Python-зависимостей для аналитического блока.

## Порядок запуска

1. Поместите скомпилированный бинарный модуль классификатора в рабочее окружение.
2. Откройте файл `SECIRA_Meteor_NRE_Classifier.ipynb` в среде Google Colab.
3. Загрузите тестовый массив геофизических данных.
4. Выполните ячейки этапа сопряжения для прогона данных через SECIRA-классификатор и генерации предсказаний классов.
5. Запустите блок статистического анализа и кластеризации для воспроизведения графиков плотности распределения и таблицы средних значений.
